# Формат транзакции

## Обычная транзакция

type => tx fixstr
from => Адрес 8 
to => Адрес 8 байт binary
amount => Amount int64
cur => тикер fixstr
timestamp => таймштамп в мс. int64
seq => Int fixint/uintXX
extradata => text (JSON) fixstr/strXX




public_key => bin pubkey
signature => bin signature
extdata => arbitrary unsigned external data

encode:
to sign make hash:
sha256(msgpack encode(
        [type, from, to, amount, cur, timestamp, seq, extradata]
        ))
Sign and make obj:
msgpack (
        {
        type => type,
        tx => [from, to, amount, cur, timestamp, seq, extradata],
        sig => {
        bin_pub_key1 => bin_signature1,
        bin_pub_key2 => bin_signature2,
        ...
        }        
        extdata => {}
        }
)
