# Формат транзакции

## Обычная транзакция

type => tx fixstr (0xa0 ... 0xbf)
from => Адрес 8 (0xc4)
to => Адрес 8 байт binary (0xc4)
amount => Amount fixint/uintXX (0x00..0x7f или 0xcc .. 0xcf)
cur => тикер fixstr (0xa0 .. 0xbf)
timestamp => таймштамп в мс. int64 (0xcf)
seq => Int fixint/uintXX (0x00..0x7f или 0xcc..0xcf)
extradata => text (JSON) fixstr/strXX (0xa0..0xbf или 0xd9..0xdb)




public_key => bin pubkey
signature => bin signature
extdata => arbitrary unsigned external data

encode:
to sign make hash:
sha256(msgpack encode(
        [type, from, to, amount, cur, timestamp, seq, extradata]
        ))
Sign and make obj:
msgpack (
        {
        type => type,
        tx => [from, to, amount, cur, timestamp, seq, extradata],
        sig => {
        bin_pub_key1 => bin_signature1,
        bin_pub_key2 => bin_signature2,
        ...
        }        
        extdata => {}
        }
)
